🖼️ Image Classification Project:

This project focuses on building and comparing different machine learning models for image classification using a subset of the Fashion-MNIST dataset.

Implemented and evaluated three models: Naive Bayes, Decision Tree, and Neural Network (MLP) for classifying images into different categories.

Preprocessed images into feature vectors and compared model performance using accuracy, precision, recall, and F1-score.

🛠️ Technologies & Tools:

Python
Scikit-learn (Naive Bayes, Decision Tree, MLP)
TensorFlow/Keras (dataset loading)
NumPy & Matplotlib
